# 🧠 Mineração de Texto - AP News

Projeto desenvolvido para análise de sentimentos e mineração de texto a partir de notícias políticas e de guerras.

## 🚀 Tecnologias
- Python
- Flask
- BeautifulSoup
- NLTK / Scikit-learn
- Chart.js

## 📊 Funcionalidades
- Coleta de dados via web scraping (AP News)
- Tradução automática dos textos
- Limpeza e normalização
- Geração de corpus em .txt
- Análise de sentimentos
- Visualização com gráficos

## 📁 Estrutura
- scraper.py
- preprocessamento.py
- corpus_final.txt
- app.py (Flask)
- templates/

## 🎯 Objetivo
Aplicar técnicas de mineração de texto para extrair insights de notícias reais.
